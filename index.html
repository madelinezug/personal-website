I'm working on...

Chirps 3.0

A new and improved website to display the announcements sent out through Pomona College's "chirps" listserve. With added functionality to search through and save announcements. This project started in my Spring 2018 Database Systems course and transitioned through my System Security course to include security features such as encrypted password storage and groups to manage authorization.

Celebration Display

A celebration display for the Pomona CS department. When a user pushes code to github, their request is routed through a raspberry pi to an FPGA that drives an LED display that displays a celebratory animation. Built for my Fall 2018 Microprocessor Systems course.
